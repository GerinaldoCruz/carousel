# Carousel

Um projeto simples de carousel/carrossel para exibir imagens.

## Estrutura do Projeto

```
├── index.html      # Arquivo HTML principal
├── app.js          # Lógica JavaScript
├── css/
│   ├── reset.css   # Reset de estilos CSS
│   └── style.css   # Estilos do carousel
└── images/         # Pasta com as imagens
```

## Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Use os controles para navegar entre as imagens

## Funcionalidades

- Navegação entre imagens
- Design responsivo
- Interface limpa e simples
